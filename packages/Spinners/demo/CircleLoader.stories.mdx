# CircleLoader

The `CircleLoader` component provides a customizable animated loading spinner with the ability to change color, size, and add a className.

## Basic Usage

You can use the `CircleLoader` like this:

## Usage

```jsx
import React from 'react';
import CircleLoader from 'react-craftify-spinners';  

const MyComponent = () => {
  return (
    <div>
      <CircleLoader />
    </div>
  );
};

```

## Custom Color

You can customize the color of the loading spinner:

```jsx
import React from 'react';
import CircleLoader from 'react-craftify-spinners';  

const MyComponent = () => {
  return (
    <div>
      <CircleLoader color="blue" />
    </div>
  );
};

```

## Custom Size

You can customize the size of the loading spinner:

```jsx
import React from 'react';
import CircleLoader from 'react-craftify-spinners';  

const MyComponent = () => {
  return (
    <div>
      <CircleLoader size="20px" />
    </div>
  );
};


```

## Custom Class Name

You can add a custom class name to the loading spinner:

```jsx
import React from 'react';
import CircleLoader from 'react-craftify-spinners';  

const MyComponent = () => {
  return (
    <div>
      <CircleLoader className="my-custom-class" />
    </div>
  );
};

```
